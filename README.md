## 🚀 GitHub Pages 호스팅 방법

### 1단계: GitHub 저장소

이미 `heystudy-legal` 저장소가 있으면 그대로 사용. 없으면 새로 만들기:

```bash

Visibility: Public
[Create repository]
```

### 2단계: 파일 업로드

옵션 A: 웹 인터페이스 (가장 쉬움)
1. 저장소 페이지에서 "Add file" → "Upload files"
2. 본 폴더의 모든 HTML 파일 드래그
3. 기존 파일이 있으면 덮어쓰기
4. "Commit changes"

옵션 B: Git 명령어
```bash
cd heystudy-legal
git pull  # 기존 저장소면
# 본 폴더 파일들을 git 저장소에 복사
git add .
git commit -m "Update legal documents to lawyer-grade quality"
git push origin main
```
